# JobTrack AI

An AI-powered job application tracking platform that helps job seekers organize applications, manage interviews, track progress, and gain insights throughout their job search.

---

## 🚀 Features (Planned)

- User Authentication (JWT)
- Dashboard & Analytics
- Job Application Tracking
- Company Management
- Interview Scheduler
- Resume & Cover Letter Management
- Search, Filter & Sort
- AI Resume Review
- AI Cover Letter Generator
- Interview Preparation

---

## 🛠️ Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Redux Toolkit
- React Router
- Material UI
- React Hook Form

### Backend

- Node.js
- Express.js
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT Authentication

### Deployment

- Frontend: Vercel
- Backend: Render
- Database: PostgreSQL (Local → Supabase)
